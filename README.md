# esp32-mmwave-radar-tripwire
A perimeter alarm built from scratch using Heltec ESP32-S3 boards, LoRa radio, and a 24GHz millimeter-wave radar sensor
This hardware is bare circuit boards. If you deploy this permanently, you MUST put the Transmitter in a waterproof plastic junction box. Millimeter-wave radar can see perfectly through plastic and glass, but it cannot see through metal. Do not put it in a metal box!
